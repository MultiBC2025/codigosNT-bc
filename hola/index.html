Guía Completa de Activación MultiTV

Configura tu cuenta fácilmente en todos tus dispositivos

Android ▼

Esto es un ejemplo de cómo se verá el contenido paso a paso.

1. Abre la aplicación IPTV Smarters Pro.

2. Selecciona 'Agregar nuevo usuario'.

3. Elige 'Iniciar sesión con Xtream Codes API'.

4. Introduce el nombre de usuario, contraseña y URL del servidor proporcionado.

5. Haz clic en 'Agregar usuario'.

iOS (iPhone & iPad) ▼

Esto es un ejemplo de cómo se verá el contenido paso a paso.

1. Descarga IPTV Smarters desde la App Store.

2. Abre la app y selecciona 'Agregar nuevo usuario'.

3. Selecciona 'Iniciar sesión con Xtream Codes API'.

4. Ingresa los datos de acceso proporcionados.

5. Pulsa en 'Agregar usuario'.

Windows ▼

Esto es un ejemplo de cómo se verá el contenido paso a paso.

1. Descarga IPTV Smarters para Windows desde el sitio oficial.

2. Instala y abre la aplicación.

3. Selecciona 'Agregar nuevo usuario'.

4. Elige 'Iniciar sesión con Xtream Codes API'.

5. Introduce el nombre de usuario, contraseña y URL del servidor.

6. Haz clic en 'Agregar usuario'.
